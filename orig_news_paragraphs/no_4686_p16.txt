A vulnerability in Microsoft Windows allowed the WannaCry hackers to strike. The WannaCry virus exploits a bug in Windows networking protocol, which Microsoft patched in March, possibly after receiving a   from the U. S. intelligence community. Some blame Microsoft for enabling this global ransomware attack with poor product design and for abandoning users running older versions of the Windows operating system. In particular, users of Windows XP, which Microsoft officially stopped supporting in 2014, were vulnerable to attack.
